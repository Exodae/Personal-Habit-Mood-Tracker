
# Personal Habit & Mood Tracker

This is a **simple, functional SQL project** for tracking personal habits and mood levels.  
It demonstrates SQL database design, sample data, and queries for insights into personal productivity and well-being.

## Tables
- `habits` – Habit names, categories, and goal frequency
- `entries` – Daily habit completion logs
- `mood_log` – Daily mood levels and notes

## Example Queries
- Count of completed habits
- Average mood over time
- Notes for low mood days
- Habit streaks

## How to Use
1. Import `schema.sql` to create tables  
2. Import `sample_data.sql` to populate sample data  
3. Run queries from `queries.sql` to explore insights
